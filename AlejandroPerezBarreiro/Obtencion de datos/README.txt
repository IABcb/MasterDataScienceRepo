EJECUCION DE LA PRACTICA

1- Descargar los datos de los ficheros stops.txt indicados en la memoria y guardarlos en formato csv.

2- Ejecutar los ficheros generarCsvCercanias.py, generarCsvMetro.py y generarCsvTranvia.py para obtener el numero de linea, de parada y el tipo de transporte y combinarlos con los datos de stops.csv generando tres nuevos csv Metro.csv, Tranvia.csv y Cercanias.csv.

3- Ejecutar el fichero unirCsv.py para juntar la informacion de los tres medios de transporte en un solo fichero Transporte.csv

4- Ejecutar el fichero crearRDF.py para crear el fichero Tranporte.xml con los datos en formato RDF/XML.
